## 🏗️ 架构特点

### ITCSS 分层架构

```
Layer 1: Variables   (设计令牌)
Layer 2: Reset       (样式重置)
Layer 3: Typography  (字体系统)
Layer 4: Layouts     (布局 - 预留)
Layer 5-6: Components (组件库)
Layer 7-8: Utilities (工具类)
```

### 命名规范

- **BEM** - 组件命名 (`.btn--primary`, `.card__header`)
- **语义化** - 变量命名 (`--theme-btn-primary-bg`)
- **Tailwind风格** - 工具类 (`.flex`, `.gap-4`, `.p-5`)

### 兼容性策略

```
✅ 100% 向后兼容
✅ 旧类名继续有效 (.primary-btn, .add-btn, .modal-btn)
✅ 新旧类名并存，平滑迁移
✅ 所有变量名保持不变 (--theme-*)
```
